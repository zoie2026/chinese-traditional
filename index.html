🏛️ 场馆PK

让我们来对比一下四种传统工艺的技法、材质和制作顺序！

工艺类型 主要材质 制作技法 制作顺序
🧱 砖雕 青砖、灰砖 浮雕、透雕、线刻 选砖 → 构思 → 雕刻 → 打磨 → 安装
💎 玉雕 玉石、翡翠、和田玉 琢磨、透雕、浮雕、镶嵌 选料 → 设计 → 粗雕 → 细琢 → 抛光
🪵 木雕 紫檀、黄花梨、红木 圆雕、浮雕、透雕、镶嵌 选材 → 绘图 → 雕刻 → 打磨 → 上漆
🏺 景泰蓝 铜胎、珐琅釉、金丝 制胎、掐丝、点蓝、烧蓝、磨光 制胎 → 掐丝 → 点蓝 → 烧蓝 → 磨光 → 镀金

🔄 传统工艺品 vs 现代工艺品

对比维度 传统工艺品 现代工艺品
制作方式 纯手工制作 机器批量生产
制作时间 耗时很长，可能数月 快速高效，几分钟完成
数量 独一无二，限量珍品 大量复制，随处可见
材质 天然材料，珍贵稀有 合成材料，成本低廉
价值 收藏价值高，文化内涵丰富 实用价值为主，价格亲民
传承意义 承载历史文化，代代相传 满足现代需求，更新换代快
💡 小思考：传统工艺品和现代工艺品各有各的美！传统工艺让我们感受到古人的智慧和匠心，现代工艺则让更多人能享受到美丽的物品。你更喜欢哪一种呢？

📝 课堂总结

本节课我们借助专题学习网站探究、小组合作绘制思维导图、趣味闯关游戏等多样形式，熟练掌握局部、整体、寓意、工艺的四步欣赏法，能从材质、造型、纹样、色彩等多角度赏析景泰蓝等传统工艺品。在小组协作、上台展示与交流评述的互动中，真切感受传统工艺独有的艺术魅力，深刻体会工匠精益求精的匠心精神与古代劳动人民的无穷智慧，懂得传统工艺是弥足珍贵的中华传统文化瑰宝，我们要珍爱传统、传承非遗。
